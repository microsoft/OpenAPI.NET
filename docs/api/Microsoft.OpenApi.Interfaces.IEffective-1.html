Interface IEffective<T>

OpenApiElements that implement IEffective indicate that their description is not self-contained. External elements affect the effective description.

Namespace: Microsoft.OpenApi.Interfaces
Assembly: Microsoft.OpenApi.dll
Syntax
public interface IEffective<T>
    where T : class, IOpenApiElement
Type Parameters
Name Description
T

Type of OpenApi Element that is being referenced.
Remarks

Currently this will only be used for accessing external references.
In the next major version, this will be the approach accessing all referenced elements. This will enable us to support merging properties that are peers of the $ref
